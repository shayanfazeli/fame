FAME: A Framework for Topic Modeling¶

Introduction¶

This repository is the codebase for the article “A Framework for Neural Topic Modeling of Text Corpora”.

Installation¶

You can install this package by running the following script:

pip3 install -e .

Or

python3 setup.py install

Usage¶

Please refer to the demo document to get started.
