# FAME: A Framework for Topic Modeling

## Introduction
This repository is the codebase for the article "[A Framework for Neural Topic Modeling of Text
Corpora](https://arxiv.org/pdf/2108.08946.pdf)".

## Installation
You can install this package by running the following script:
```
pip3 install -e .
```

Or
```
python3 setup.py install
```

## Usage
Please refer to the [demo document](https://github.com/shayanfazeli/fame/blob/master/notebooks/20newsgroups/demo_fame.ipynb) to get started.
